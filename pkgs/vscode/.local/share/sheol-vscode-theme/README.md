# Sheol Dark — VS Code

The Tarnished Reliquary as a code editor theme. Strict warm palette: gold on abyss-black, no cool tones, deliberate restraint.

Part of the [SHEOL rice](https://github.com/jordanlanham52/arch-dot-files).

## Palette

- `#050507` abyss — editor background
- `#b8aea0` bone — primary text
- `#a08240` gilt — keywords, emphasis
- `#c9a651` leaf — strings, types
- `#e8c870` halo — accents, cursor
- `#5a1a1a` sanctus — errors, deletions
- `#4a3a1f` tarnish — comments
- `#6b6470` linen — secondary text
